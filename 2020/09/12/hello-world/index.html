第一个博客，虽然大多是抄的，但总体感觉，还是挺简单的！

hexo安装

  • npm install -g hexo:安装hexo

  • cd hexo

  • hexo init:初始化 ### hexo常用命令

  • hexo n name：创建新的博客

  • hexo clean:清除缓存等

  • hexo g:生成html文件

  • hexo s:本地预览

  • hexo d:提交
